Guía rápida

Uso en GitHub Pages

school Alumno

Entra con el rol Alumno. Solo visualiza la lista de CVU, las horas reales, la meta semanal ajustada, el detalle por día y el estatus calculado por semana.

admin_panel_settings Coordinador

Entra con rol Coordinador, carga el CSV de referencia, el archivo .dat, registra calendario institucional y exporta el JSON público para actualizar GitHub Pages.

calendar_month Seguimiento semanal

El sistema separa los marcajes por semana real, de lunes a domingo. La semana evaluada se determina por las fechas internas del archivo .dat, no por la fecha actual del usuario. La meta se calcula de lunes a sábado y se ajusta si hay asuetos, inhábiles o vacaciones.

security Límite técnico

GitHub Pages no ejecuta backend ni base de datos. La clave de coordinación es una barrera local básica, no autenticación real. Para seguridad institucional se requiere servidor o servicio externo.
